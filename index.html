Version Control with Ida

git init
Creates a hidden git directory and enables version control on that project.
git clone url
Copies the supplied url from github to your local machine
git status
Shows the current differences between local and branch.
git pull
updates your local repo to be the same as the remote repo
git add "file"

Adds specfied files to the staging ground to prepare for a commit

Use "git add ." to add all files.

git commit -m "message"

Commits the file to branch on local machine (your own computer) and specifies a commit message.

Describe the things you have changed briefly in the message.

git push
Pushes the local commit to the branch on origin (the repository on github).
git branch
Displays all branches of the repository.
git checkout "branch name"
Changes to the specified branch.
git checkout -b "branch name"
Creates a branch with the specified name and changes to it.
